# Встречи с преподавателями

Веб-приложение на Flask для онлайн-записи родителей на встречи с преподавателями и просмотра заявок преподавателями.

## Как запустить

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Сайт откроется на `http://127.0.0.1:5000/`.

## Вход в систему
- Администратор: `admin@school.local` / `admin123`
- Родитель: `parent@example.com` / `parent123`
- Преподаватель (пример): `1@school.local` / `teacher123` (номер в email соответствует id преподавателя)

Родители видят список преподавателей и могут бронировать слоты. Преподаватель видит только своё расписание и записи родителей. Администратор управляет учетками (создает/меняет/удаляет).

## Возможности
- Разделение по ролям: родитель и преподаватель.
- Список преподавателей с ФИО, предметом, фото и номером кабинета.
- Выбор временных слотов, отправка ФИО, email и вопроса; слоты сохраняются в SQLite и становятся занятыми.
- Панель преподавателя с таблицей записей родителей и их временем.
